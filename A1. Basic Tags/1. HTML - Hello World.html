  2. Tambahkan "li" element untuk membuat list
  3. Urutan list dimulai dari angka 1 dan seterusnya
    1. Membuat List Bersarang
    2. Ordered & Ordered List bisa bersarang
    3. Tambahkan lagi ol element di dalam ol element
    4. Tambahkan li element untuk membuat element
